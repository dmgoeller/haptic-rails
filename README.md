# Haptic Rails

## Components

### Badges

```html
<span class="haptic-badge">Text</span>
<span class="haptic-badge primary">Text</span>
<span class="haptic-badge secondary">Text</span>
<span class="haptic-badge red">Text</span>
<span class="haptic-badge green">Text</span>
```
